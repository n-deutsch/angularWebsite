DeutschAudio

A music playing app for Android phones and tablets. Uses circular linked lists to play audio files stored on the device. Created from scratch with Java and Android Studio.

Github: https://github.com/n-deutsch/DeutschAudio

SDK: DeutschAudio

Kangaroo Punch

A C# library that uses OpenExchange and CryptoCompare APIs to calculate real-time currency exchanges. Works with world currencies and cryptocurrencies. Created by Alex Elwell and Nathan Deutsch for the 2018 WSU Hackathon.

Github: https://github.com/n-deutsch/Kangaroo_Punch

Linear Algebra Companion

A desktop application created in Python to assist with linear algebra problems. Handles matrix reduction, multiplication, addition, and subtraction. UI Created with Tkinter. Includes a "test.py" file for unit and integration tests.

Github: https://github.com/n-deutsch/LinearAlgebraCompanion

Spacegame

A city building game created in C# that I will probably never finish.

Github: https://github.com/n-deutsch/SPACEGAME

ME401 Robot

Software for a robot that navigates a maze and chooses the correct parking space by perceiving color. You can't run this without incredibly specific hardware, I just wanted to show it off. Created by Jared Meboe and Nathan Deutsch for Mechanical Engineering 401.

Github: https://github.com/n-deutsch/ME401Robot
